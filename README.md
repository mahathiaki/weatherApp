This is a simple and user-friendly Weather App built using HTML, CSS, and JavaScript. The app lets users search for any city and view live weather details like temperature, humidity, wind speed, and weather conditions in real time using the OpenWeatherMap API. Different weather icons are displayed based on the current climate conditions to make the interface more interactive and easy to understand. The project also handles invalid city names properly and provides a clean, responsive design that works well on different screen sizes. Through this project, concepts like API integration, asynchronous JavaScript, DOM manipulation, and responsive web design were implemented and practiced.
